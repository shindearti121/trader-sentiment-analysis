# Trader Behavior vs Market Sentiment Analysis

## Objective
Analyze how market sentiment impacts trader behavior and performance.

## Methodology
- Data cleaning and preprocessing
- Feature engineering (PnL, win rate, leverage, trade frequency)
- Sentiment-based comparison
- Trader segmentation

## Key Insights
1. Trader performance improves during Greed periods.
2. Traders reduce risk exposure during Fear periods.
3. High leverage trading leads to higher volatility in outcomes.

## Strategies
- Reduce leverage during Fear phases
- Increase participation during Greed with controlled risk
- Align trading strategies with sentiment conditions

## How to Run
1. Install dependencies: pandas, matplotlib, seaborn
2. Run the notebook: `analysis.ipynb`
